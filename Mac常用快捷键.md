# 常用快捷键

*默认快捷键*

|           效果            |                快捷键                 |
| :-----------------------: | :-----------------------------------: |
|  显示 / 隐藏隐藏文件(夹)  |         `command + shift + .`         |
|      窗口全屏 / 还原      |         `command + ctrl + F`          |
|        窗口最小化         |             `command + M`             |
|       关闭当前窗口        |             `command + W`             |
|       退出当前应用        |             `command + Q`             |
| 文件 / 文件夹剪切（移动） | `command + C`, `command + option + V` |
